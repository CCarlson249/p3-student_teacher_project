# p3-student_teacher_project
This is a college student and class management system. This system utilizies 4 classes (students, teachers, classes, and reviews). This system allows students to enroll and drop classes and leave reviews for their teachers. Teachers can give students grades choose their classes. Classes will contain a teacher and multiple students and their grades.
